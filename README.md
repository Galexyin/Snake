# Snake
The Simple Snake Game
=====================
This is a Simple Snake game using C.Graphic Libreries are not Used.
Version 1.0
============
-Simple Interface
-Canvas is non resizable
-Speed is Changable

Version 2.0
============
-Canvas is Resizable.
-Speed indicator added.

Version 3.0
============
-HighScore option added.

 Simple Build using GCC or MigWC.
